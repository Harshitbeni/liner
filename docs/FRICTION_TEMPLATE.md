# Friction log entry template

Copy into [DOGFOOD.md](./DOGFOOD.md) friction log (items 1–3).

```markdown
### N. Short title

- **What happened:** (steps you took, what you saw)
- **Expected:** (what should have happened)
- **Severity:** low / medium / high
- **Area:** outline / thread / harness / craft / desktop / other
```

Keep each entry to 2–4 sentences. Link follow-up issues in the **Follow-ups** table at the bottom of DOGFOOD.md.
